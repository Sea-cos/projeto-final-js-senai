Matrizes [ ][ ]

Matrizes são arrays bidimensionais que representam dados em linhas e colunas - a base para grids, tabelas e dados tabulares

Resumo

Uma matriz em JavaScript é simplesmente um array cujos elementos também são arrays - um array de arrays. Para acessar um elemento, usa-se dois indices: o primeiro seleciona a linha e o segundo seleciona a coluna: matriz[linha][coluna].

Matrizes são mmuito utilizadas para representar grades de jogos, planilhas, imagens pixel-a-pixel, grafos de adjacência e qualquer dado com estrutura tabular. Para percorrê-las normalmente usa-se dois loops for aninhados.

Tópicos

Crie uma matriz como um array literal de arrays. Para percorrer todos os elementos, use dois for aninhados - o externo para linhas e o interno para as colunas.

A propriedade matriz.length retorna o número de linhas, matriz[0].length retorna o numero de colunas da primeira linha (assumindo que todas as linhas têm o mesmo tamanho).

JavaScript 
const matriz = [
     [1, 2, 3],
     [4, 5, 6],
     [7, 8, 9]
];
                                    
console.log(matriz[1][2]); // 6 (linha 1, coluna 2)
                                    
for (let i = 0; i < matriz.length; i++) {
    for (let j = 0; j < matriz[i].length; j++) {
        console.log(matriz[i][j]);
    }
}
