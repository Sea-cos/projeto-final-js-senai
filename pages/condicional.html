Estruturas Condicionais

Condicionais permitem que seu programa tome decisões com base em expressões que resultam em verdadeiro ou falso.

Resumo

Uma estrutura condicional avalia uma expressão e executa diferentes blocos de código conforme o resultado. São a base do controle de fluxo em qualquer programa.

Use if...else para condições booleanas e fluxos com poucas ramificações. Use switch quando um único valor precisa ser comparado com muitas opções distintas, tornando o código mais legível.

Tópicos

O if avalia uma condição. Se for verdadeiro, executa o bloco. O else if encadeia outras condições. O else captura tudo que não foi capturado antes. Você pode encadear quantos else if precisar.

JavaScript 
let nota = 75;

if (nota >= 90) {
    console.log("A");
} else if (nota >= 70) {
    console.log("B");
} else {
    console.log("C");
}

Exemplo 1

Faça um algoritmo que leia um número inteiro e diga se ele é par ou ímpar

JavaScript 
function verificar() {
    let numero = document.getElementById("numero").value;

    if (numero === "") {
        document.getElementById("resultado").innerText = "Digite um número!";
    } else {
        numero = parseInt(numero);

        if (numero % 2 === 0) {
            document.getElementById("resultado").innerText = "O número é PAR";
        } else {
            document.getElementById("resultado").innerText = "O número é ÍMPAR";
        }
    }
}

O switch compara uma expressão com uma série de cases usando igualdade estrita (===). Ao encontrar um match, executa o bloco daquele caso.

O break é essencial para evitar o fall-through (execução de cases seguintes mesmo sem match). O default funciona como o else: é executado quando nenhum case corresponder.

JavaScript 
let dia = "segunda";

switch (dia) {
  case "segunda":
    console.log("Início da semana");
    break;
  case "sexta":
    console.log("Quase fim de semana!");
    break;
  default:
    console.log("Dia normal");
}
