Vetores [ ]

Arrays são coleções ordenadas de elemtnos que permitem armazenar e manipular lista de dados de forma eficiente.

Resumo

Um vetor (ou array unidimensional) é uma estrutura de dados que armazena uma sequência de valores em posições numeradas, chamadas índices. Em JavaScript, arrays são objetos dinâmicos: podem conter qualquer tipo de dado e crescer ou diminuir durante a execução.

Os índices começam em 0 (zero-indexed). O array ["a", "b", "c"] tem arr[0] = "a", arr[1] = "b" e arr[2] = "c". A propriedade .length retorna o número de elementos.

Tópicos

Arrays são criados com colchetes [] ou com new Array(). Acesse elementos pelo índice: arr[0]. Modifique com atribuição direta: arr[1] = "novo"

Os métodos mais usados incluem push (adiciona ao fim), pop (remove do fim), shift (remove do início), unshift (adiciona ao inicio), map (transforma cada elemento) e filter (filtra elementos).

JavaScript 
const frutas = ["maçã", "banana", "uva"];
                                    
console.log(frutas[0]);   // "maçã"
console.log(frutas.length); // 3
                                    
frutas.push("manga");
// frutas = ["maçã", "banana", "uva", "manga"]
                                    
const maiusculas = frutas.map(f => f.toUpperCase());
