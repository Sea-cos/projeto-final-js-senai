Estruturas de Repetição

Loops permitem executar um bloco de código múltiplas vezes. Em JavaScript existem três formas principais, cada uma com seu caso de uso ideal.

Resumo

Uma estrutura de repetição (ou laço) executa um trecho de código enquanto uma condição for verdadeira, ou por um número determinado de vezes. São essenciais para percorrer listas, processar dados e automatizar tarefas repetitivas.

Use for quando souber o número de iterações antecipadamente, while quando a condição de parada for avaliada antes de cada execução, e do...while quando o bloco precisar executar pelo menos uma vez.

Tópicos

O laço for é ideal quando você sabe exatamente quantas vezes quer repetir um bloco. Ele possui três partes: inicialização (executada uma vez no começo), condição (verificada antes de cada iteração) e incremento (executado ao final de cada iteração).

Sintaxe: for (inicialização; condição; incremento) { ... }

JavaScript 
for (let i = 0; i < 5; i++) {
  console.log(i);
}

O laço while continua executando enquanto a condição for verdadeira. A condição é avaliada antes de cada execução do bloco, portanto se for falsa desde o início, o bloco nunca será executado.

Cuidado com loops infinitos: certifique-se de que a condição eventualmente se tornará falsa.

JavaScript 
let i = 0;
while (i < 5) {
  console.log(i);
  i++;
}

O laço do...while é semelhante ao while, mas a condição é verificada depois da execução do bloco. Isso garante que o código dentro do bloco seja executado pelo menos uma vez, independentemente da condição.

Útil quando precisa executar uma ação antes de verificar se deve continuar — como validar input de usuário.

JavaScript 
let i = 0;
do {
  console.log(i);
  i++;
} while (i < 5);
