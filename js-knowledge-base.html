Estruturas

Estruturas de Repetição

Loops permitem executar um bloco de código múltiplas vezes. Em JavaScript existem três formas principais, cada uma com seu caso de uso ideal.

Resumo

Uma estrutura de repetição (ou laço) executa um trecho de código enquanto uma condição for verdadeira, ou por um número determinado de vezes. São essenciais para percorrer listas, processar dados e automatizar tarefas repetitivas.

Use for quando souber o número de iterações antecipadamente, while quando a condição de parada for avaliada antes de cada execução, e do...while quando o bloco precisar executar pelo menos uma vez.

for while do...while iteração loop infinito break / continue
Tópicos

O laço for é ideal quando você sabe exatamente quantas vezes quer repetir um bloco. Ele possui três partes: inicialização (executada uma vez no começo), condição (verificada antes de cada iteração) e incremento (executado ao final de cada iteração).

Sintaxe: for (inicialização; condição; incremento) { ... }

JavaScript 
// Adicione seu exemplo aqui
for (let i = 0; i < 5; i++) {
  console.log(i);
}

O laço while continua executando enquanto a condição for verdadeira. A condição é avaliada antes de cada execução do bloco, portanto se for falsa desde o início, o bloco nunca será executado.

Cuidado com loops infinitos: certifique-se de que a condição eventualmente se tornará falsa.

JavaScript 
// Adicione seu exemplo aqui
let i = 0;
while (i < 5) {
  console.log(i);
  i++;
}

O laço do...while é semelhante ao while, mas a condição é verificada depois da execução do bloco. Isso garante que o código dentro do bloco seja executado pelo menos uma vez, independentemente da condição.

Útil quando precisa executar uma ação antes de verificar se deve continuar — como validar input de usuário.

JavaScript 
// Adicione seu exemplo aqui
let i = 0;
do {
  console.log(i);
  i++;
} while (i < 5);

Estruturas Condicionais

Condicionais permitem que seu programa tome decisões com base em expressões que resultam em verdadeiro ou falso.

Resumo

Uma estrutura condicional avalia uma expressão e executa diferentes blocos de código conforme o resultado. São a base do controle de fluxo em qualquer programa.

Use if...else para condições booleanas e fluxos com poucas ramificações. Use switch quando um único valor precisa ser comparado com muitas opções distintas, tornando o código mais legível.

if else if else switch case default break operador ternário
Tópicos

O if avalia uma condição. Se for truthy, executa o bloco. O else if encadeia outras condições. O else captura tudo que não foi capturado antes. Você pode encadear quantos else if precisar.

O operador ternário (condição ? valorSe : valorNão) é uma versão compacta do if...else para expressões simples.

JavaScript 
// Adicione seu exemplo aqui
let nota = 75;

if (nota >= 90) {
  console.log("A");
} else if (nota >= 70) {
  console.log("B");
} else {
  console.log("C");
}

O switch compara uma expressão com uma série de cases usando igualdade estrita (===). Ao encontrar um match, executa o bloco daquele caso.

O break é essencial para evitar o fall-through (execução de cases seguintes mesmo sem match). O default funciona como o else: é executado quando nenhum case corresponder.

JavaScript 
// Adicione seu exemplo aqui
let dia = "segunda";

switch (dia) {
  case "segunda":
    console.log("Início da semana");
    break;
  case "sexta":
    console.log("Quase fim de semana!");
    break;
  default:
    console.log("Dia normal");
}
Estruturas de Dados

Vetores [ ]

Arrays são coleções ordenadas de elementos que permitem armazenar e manipular listas de dados de forma eficiente.

Resumo

Um vetor (ou array unidimensional) é uma estrutura de dados que armazena uma sequência de valores em posições numeradas, chamadas índices. Em JavaScript, arrays são objetos dinâmicos: podem conter qualquer tipo de dado e crescer ou diminuir durante a execução.

Os índices começam em 0 (zero-indexed). O array ["a", "b", "c"] tem arr[0] = "a", arr[1] = "b" e arr[2] = "c". A propriedade .length retorna o número de elementos.

push / pop shift / unshift splice / slice indexOf / includes map / filter / reduce forEach sort / reverse
Tópicos

Arrays são criados com colchetes [] ou com new Array(). Acesse elementos pelo índice: arr[0]. Modifique com atribuição direta: arr[1] = "novo".

Os métodos mais usados incluem push (adiciona ao fim), pop (remove do fim), shift (remove do início), unshift (adiciona ao início), map (transforma cada elemento) e filter (filtra elementos).

JavaScript 
// Adicione seu exemplo aqui
const frutas = ["maçã", "banana", "uva"];

console.log(frutas[0]);   // "maçã"
console.log(frutas.length); // 3

frutas.push("manga");
// frutas = ["maçã", "banana", "uva", "manga"]

const maiusculas = frutas.map(f => f.toUpperCase());

Matrizes [ ][ ]

Matrizes são arrays bidimensionais que representam dados em linhas e colunas — a base para grids, tabelas e dados tabulares.

Resumo

Uma matriz em JavaScript é simplesmente um array cujos elementos também são arrays — um array de arrays. Para acessar um elemento, usa-se dois índices: o primeiro seleciona a linha, e o segundo seleciona a coluna: matriz[linha][coluna].

Matrizes são muito utilizadas para representar grades de jogos, planilhas, imagens pixel-a-pixel, grafos de adjacência e qualquer dado com estrutura tabular. Para percorrê-las, normalmente usa-se dois loops for aninhados.

array de arrays matriz[i][j] loops aninhados linhas e colunas matriz quadrada transposta
Tópicos

Crie uma matriz como um array literal de arrays. Para percorrer todos os elementos, use dois for aninhados — o externo para as linhas e o interno para as colunas.

A propriedade matriz.length retorna o número de linhas. matriz[0].length retorna o número de colunas da primeira linha (assumindo que todas as linhas têm o mesmo tamanho).

JavaScript 
// Adicione seu exemplo aqui
const matriz = [
  [1, 2, 3],
  [4, 5, 6],
  [7, 8, 9]
];

console.log(matriz[1][2]); // 6 (linha 1, coluna 2)

for (let i = 0; i < matriz.length; i++) {
  for (let j = 0; j < matriz[i].length; j++) {
    console.log(matriz[i][j]);
  }
}
